# Company-Specific Guides

Each company runs FDE interviews differently. These guides cover what's unique about each company's process.

| Company | File | Key differentiator |
|---------|------|-------------------|
| Google | [google-fde.md](google-fde.md) | Two distinct rounds: DSA + RRK. GCP depth critical. |
| Anthropic | [anthropic.md](anthropic.md) | AI Engineering depth. Constitutional AI. Claude API. |
| Meta | [meta-fde.md](meta-fde.md) | Advertising tech. Llama ecosystem. Scale. |
| Amazon | [amazon-fde.md](amazon-fde.md) | Leadership Principles heavily weighted. AWS depth. |
| Microsoft | [microsoft-fde.md](microsoft-fde.md) | Azure AI. Copilot ecosystem. More advisory than code. |

Use these guides alongside the general resources in sections 01-04. The company-specific guides tell you what to emphasize, not what to skip.

# Books and Courses Worth Your Time

Curated. Opinionated. Only the ones that actually matter.

---

## DSA

### Books

**"Cracking the Coding Interview" — Gayle Laakmann McDowell**  
The standard. Chapter 1-7 for foundations. Chapter 16-17 for hard problems. Don't try to read all 189 problems — pick by pattern.

**"Elements of Programming Interviews in Python" — Aziz, Lee, Prakash**  
More challenging than CTCI. Better for engineers targeting senior roles. Use after you've built the foundations.

### Courses

**NeetCode on YouTube/NeetCode.io (Free)**  
Best free DSA resource. Pattern-based, Python solutions, excellent explanations. Start here.

**Grokking the Coding Interview — DesignGurus.io**  
Pattern-based curriculum. Good for engineers who don't want to figure out the patterns themselves.

## AI Engineering

### Courses

**Vizuara Minor in GenAI** (Neeraj's current course)  
Deep, practical AI Engineering curriculum. Covers LLMs, RAG, fine-tuning, and deployment.

**DeepLearning.AI Short Courses (Free)**  
- Building Systems with the ChatGPT API
- LangChain for LLM Application Development
- Building and Evaluating Advanced RAG
- Fine-tuning Large Language Models
All free. All worth doing.

**Fast.ai Practical Deep Learning (Free)**  
If you want to understand neural networks from first principles. Practical, Python-first.

### Books

**"Designing Machine Learning Systems" — Chip Huyen**  
The best book on production ML engineering. Essential reading for any FDE targeting AI roles.

**"Building LLMs for Production" — Louis-François Bouchard**  
Practical guide to building real LLM applications. Good for engineers who want concrete implementation patterns.

---

## System Design

### Books

**"System Design Interview" — Alex Xu (Volumes 1 and 2)**  
The standard. Clear explanations, good diagrams. Not perfect on depth but excellent for interview preparation.

**"Designing Data-Intensive Applications" — Martin Kleppmann**  
The definitive book on distributed systems. Not interview-specific but gives you the depth that distinguishes great system design answers.

### Courses

**ByteByteGo (Alex Xu's course)**  
Video companion to the System Design Interview book. Good visual explanations.

## GCP

**Google Cloud Skills Boost**  
Hands-on labs, official Google content. Free trial available. Work through the Cloud Architect and ML Engineer paths.

**Coursera: Google Cloud Professional Data Engineer**  
Best online course specifically for the Professional Data Engineer certification.

## Career and Behavioral

**"The Tech Resume Inside Out" — Gergely Orosz**  
The best resource on engineering resumes, specifically for senior engineers targeting FAANG.

**Gergely Orosz / The Pragmatic Engineer (Newsletter)**  
If you're targeting FAANG, subscribe. Covers hiring, career growth, and the industry accurately.

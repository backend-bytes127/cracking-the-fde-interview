## How to Use This List

**Don't grind randomly.** Work through patterns in the [02-dsa/patterns](../02-dsa/patterns/) section first. Then use this list to practice applying each pattern.

**Recommended order:**
1. Arrays + Two Pointers (1 week)
2. Sliding Window + Prefix Sum (1 week)
3. Binary Search (3 days)
4. Trees + BFS/DFS (1.5 weeks)
5. Graphs + Topological Sort (1 week)
6. DP (1.5 weeks)
7. Heap + Stack + Design (1 week)

**After each problem:** Explain the pattern out loud. If you can't say "this is [pattern] because [trigger]", you don't know it well enough yet.

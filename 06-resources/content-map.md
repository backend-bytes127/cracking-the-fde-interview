# Content Map — What to Study and When

This is the preparation roadmap. Choose your timeline, follow the priorities.

---

## 12-Month Plan (Neeraj's Plan)

### Months 1-2: Foundations

**DSA:**
- Work through all 11 pattern guides in [02-dsa/patterns](../02-dsa/patterns/)
- Solve 3-4 problems per day from the curated list
- Start recording solo mock interviews (2x/week)

**AI Engineering:**
- Complete Vizuara Minor in GenAI
- Build a RAG pipeline from scratch (not a tutorial — build it yourself)
- Read the Anthropic Constitutional AI paper

**GCP:**
- Start Google Cloud Skills Boost — Cloud Architect learning path
- Get hands-on with BigQuery (run queries on public datasets)

### Months 3-4: Depth

**DSA:**
- Hard problems — LRU Cache, Merge K Sorted Lists, Largest Rectangle
- Pramp 2x/week
- Focus on the patterns you're weakest in

**AI Engineering:**
- Deploy a model to Vertex AI
- Build and evaluate a RAG system
- Read "Designing Machine Learning Systems" (Chip Huyen)

**Certifications:**
- Complete Anthropic API Fundamentals
- Start studying for Cloud Digital Leader

**GCP:**
- Hands-on with Vertex AI, Cloud Run, Pub/Sub

### Months 5-6: Integration

**DSA:**
- 50 mock interviews milestone
- Real-world problems (02-dsa/real-world-problems)
- Review weakest pattern areas

**System Design:**
- Work through all system-design files in [04-rrk/system-design](../04-rrk/system-design/)
- Practice explaining architectures out loud

**Certifications:**
- Cloud Digital Leader exam
- Anthropic Prompt Engineering cert

### Months 7-9: Specialization

**GCP Deep Dive:**
- Professional ML Engineer certification path
- Advanced Vertex AI: fine-tuning, pipelines, Model Registry

**Company Research:**
- Deep dive on [05-company-specific/google-fde.md](../05-company-specific/google-fde.md)
- Research current Google Cloud products and recent launches

**Mock Interviews:**
- interviewing.io session (paid)
- Pramp 3x/week

### Months 10-12: Final Preparation

**Certifications:**
- Professional ML Engineer exam
- Start Professional Data Engineer path

**Mock Interviews:**
- 2-3 paid sessions
- All weak areas addressed

**Application:**
- Update resume
- Connect with Google FDE engineers on LinkedIn
- Apply

## The Minimum (3 Weeks Before Interview)

If you have 3 weeks and a scheduled interview:

1. **DSA pattern map** — memorize the triggers and templates
2. **Mock interview guide** — practice the communication framework
3. **Target company guide** — know their specific RRK topics
4. **Book 3 mock interviews** — one per week
5. **GCP quickstart** — hands-on with Vertex AI and BigQuery
6. **5 behavioral stories** — written out, practiced out loud

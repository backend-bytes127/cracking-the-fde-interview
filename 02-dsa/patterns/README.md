# DSA Patterns

Pattern recognition is the skill. Not memorization.

Every pattern guide follows the same format:
- **Trigger** — what in the problem tells you to use this pattern
- **Core idea** — one sentence
- **Template** — Python code you can adapt
- **Problems** — 3 worked examples
- **FDE insight** — how this pattern appears in real-world FDE contexts

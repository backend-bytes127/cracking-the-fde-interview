# GenAI on Cloud

The major cloud providers each have a GenAI stack. FDE candidates — especially for Google roles — need to know these deeply.

| File | Description |
|------|-------------|
| [vertex-ai.md](vertex-ai.md) | Google Cloud — Vertex AI, Model Garden, Gemini |
| [bedrock.md](bedrock.md) | AWS — Amazon Bedrock, SageMaker |
| [azure-openai.md](azure-openai.md) | Microsoft Azure — Azure OpenAI Service |

## Quick Comparison

| Platform | Key models | Strengths | When to recommend |
|----------|-----------|-----------|-------------------|
| Vertex AI | Gemini, PaLM, open-source | GCP integration, enterprise, Gemini | Google accounts, GCP-first |
| Bedrock | Claude, Llama, Titan, Mistral | AWS integration, model variety | AWS accounts, multi-model |
| Azure OpenAI | GPT-4, o-series | Microsoft ecosystem | Microsoft 365, Azure accounts |
